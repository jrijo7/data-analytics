Este repositório contém as resoluções das listas de exercícios, desafios e os códigos de exemplos práticos desenvolvidos durante a formação em Data Analytics.

## Estrutura do Repositório

O repositório está organizado em módulos, onde cada pasta representa um assunto específico da formação. Dentro de cada módulo, você encontrará os notebooks (`.ipynb`) correspondentes às aulas e as minhas soluções completas das listas propostas:

```text
data-analytics/
├── 1_fundamentos_python/

│   ├── exercicio_3_4_calculadora_desconto.ipynb
│   ├── exercicio_4_3_validador_senha.ipynb
│   └── exercicio_4_4_menu_interativo.ipynb
├── 2_coletando_os_dados/

│   ├── aula-2.ipynb
│   └── ...
├── 3_tratando_os_dados/
│   ├── aula-1.ipynb
│   ├── aula-2.ipynb
│   └── ...
└── ...

```

Cada notebook contém o código original, as explicações práticas desenvolvidas durante as aulas do respectivo módulo e as seções dedicadas às resoluções detalhadas de cada desafio.

## 🛠️ Tecnologias Utilizadas

* **Python 3** (Lógica de programação, estruturas condicionais e loops)
* **Jupyter Notebook / Google Colab** (Ambiente interativo para desenvolvimento e análise de dados)
